Bem-vindo à Khaireluna Tech

Na Khaireluna Tech você encontra materiais criados para quem quer aprender programação de forma prática.

Programação para Iniciantes

O E-book Programação para Iniciantes ensina lógica e algoritmos passo a passo, preparando você para entender qualquer linguagem de programação.

Programação com JavaScript

O E-book Programação com JavaScript mostra como usar a linguagem que dá vida à web, criando páginas interativas e dinâmicas com exemplos aplicáveis.

Programação com Python

O E-book Programação com Python apresenta desde a sintaxe básica até aplicações em automação e análise de dados, mostrando como usar Python em diferentes cenários.

Curso HTML5 e CSS3

O Curso HTML5 e CSS3 ensina a estruturar e estilizar páginas modernas, com foco em responsividade e boas práticas de desenvolvimento front-end.

Registro de Domínios

Também oferecemos o serviço de Registro de Domínios, garantindo que você tenha um endereço próprio na internet, com suporte profissional para servidor e CNPJ.
